## 📖 简介 (Introduction)

**TraeSkill** 是一个标准化的 **Prompt Context** 与 **AI 技能集合**，旨在通过 Trae IDE 的 `.trae/Skills` 机制，将行业最佳实践注入到 AI 辅助编程的工作流中。

当你在 Trae 中使用此库时，AI 将不再是一个通用的聊天机器人，而是瞬间变身为精通 **React 性能优化**、**Flutter 架构设计**、**产品需求分析** 以及 **自动化测试** 的专家伙伴。

## 🌟 核心能力 (Core Capabilities)

此仓库包含经过验证的工程化 Prompt 和上下文规则，覆盖软件开发全生命周期：

### 🧠 Product & Design (产研设计)
- **Product Manager**: 专业的 PRD 生成、用户故事拆解与头脑风暴引导。
- **UI/UX Designer**: 基于组件库的前端实现规范与 Web 设计指南。

### 🏗 Architecture & Backend (架构与后端)
- **System Architect**: 领域驱动设计 (DDD)、RESTful/GraphQL API 设计规范。
- **Backend Engineering**: Node.js 与 Python 服务端模式、数据库迁移策略、SQL 性能优化。

### 💻 Frontend & Mobile (大前端)
- **React Specialist**: 深度集成的 React 渲染优化、Hooks 最佳实践、状态管理策略。
- **Flutter Expert**: Flutter 整洁架构 (Clean Architecture)、性能分析与状态管理。

### 🛡 Quality Assurance (质量保证)
- **Testing Automation**: 基于 Playwright/Selenium 的浏览器自动化测试模式。
- **Security**: 安全需求提取与代码审计规则。

## 🔧 技术原理 (How it Works)

TraeSkill 利用了 Trae IDE 的原生 **Skills** 特性。所有规则文件均位于 `.trae/Skills` 目录下。当你在 Trae 中打开此项目或将其配置为全局技能时，Trae 的 AI Agent 会自动索引这些 Markdown 文件，并在你提问相关问题（如“优化这个 React 组件”）时，自动检索并应用 `03_Developer_ReactBestPractices` 中的规则。

## 📦 快速开始 (Quick Start)

1.  **克隆仓库**:
    ```bash
    git clone https://github.com/your-username/TraeSkill.git
    ```
2.  **在 Trae 中打开**:
    直接使用 Trae 打开此文件夹。
3.  **开始对话**:
    尝试问 AI：“帮我设计一个用户登录系统的 API” 或 “检查这段代码的渲染性能”。
