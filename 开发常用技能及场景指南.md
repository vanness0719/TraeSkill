# Trae 常用开发技能及场景指南 🚀

本指南旨在帮助开发者快速了解 Trae Skills 库中**最常用、最容易触发**的技能，并提供实际开发中的应用场景，让你像指挥专家团队一样进行协作。

---

## 🧩 0. 任务调度与编排 (Orchestration)

当你有一个复杂的、跨领域的、包含多个步骤的任务，不确定从哪开始时使用。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **任务调度专家 (Dispatcher)** | `分配任务`, `拆解需求`, `自动化流程` | "我想做一个自动找直播领福袋的系统，包含采集、记录和看板。" |

---

## 🏗 1. 需求与规划 (Idea to Plan)

当你只有一个模糊的想法，或者需要梳理复杂的业务逻辑时使用。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **全能开发团队** | `走一遍流程`, `从头开发`, `项目启动` | "我有个做个人记账 App 的想法，请带我走一遍全流程。" |
| **产品经理 (PM)** | `头脑风暴`, `梳理需求`, `竞品分析` | "针对这个 AI 聊天功能，帮我做一次深度头脑风暴，挖掘潜在需求。" |
| **API 设计专家** | `设计接口`, `RESTful`, `GraphQL` | "帮我设计一套符合最佳实践的用户权限管理系统的 REST API。" |

---

## 🎨 2. UI/UX 与前端落地 (Design to UI)

当你的页面“不够专业”，或者需要将设计稿转化为高质量代码时使用。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **Web 设计指南** | `UI 审计`, `视觉优化`, `页面太丑` | "检查一下我的登录页面，从 UI 规范和易用性角度给出修改建议。" |
| **前端落地专家** | `实现组件`, `切图`, `还原设计` | "参考 shadcn/ui 的风格，帮我写一个复杂的仪表盘侧边栏组件。" |
| **Artifacts 构建器** | `写个网页`, `交互 demo`, `单页应用` | "帮我写一个带实时搜索和分页功能的交互式数据表格 Artifact。" |

---

## 💻 3. 核心开发 (Implementation)

日常编码中针对特定框架和性能优化的专家支持。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **React/Next.js 专家** | `性能优化`, `重构 React`, `水合报错` | "这段 Next.js 代码加载太慢了，请按 Vercel 性能规范进行重构。" |
| **Flutter 专家** | `手机 App`, `Flutter 架构`, `状态管理` | "用整洁架构 (Clean Architecture) 帮我实现 Flutter 的登录逻辑。" |
| **Node.js 专家** | `Node 后端`, `中间件`, `并发处理` | "帮我重构这个 Node.js 异步处理逻辑，确保高并发下的稳定性。" |
| **Python 专家** | `FastAPI`, `异步爬虫`, `数据处理` | "用 FastAPI 写一个支持异步文件上传的后端接口。" |

解决数据存储、查询性能和迁移问题。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **数据库专家** | `优化 SQL`, `表结构设计`, `数据迁移` | "这个查询在 10 万条数据下很慢，帮我优化 SQL 并建议索引。" |
| **MCP 构建器** | `写个 MCP`, `连接外部工具`, `扩展 Trae` | "我想让 Trae 能读取我的本地备忘录，帮我写一个 MCP Server。" |

## 📂 6. 办公与通用 (Utilities)

处理非代码类的文档和安全审计。

| 技能名称 | 唤醒关键词 | 示例场景 |
| :--- | :--- | :--- |
| **安全专家** | `安全审计`, `漏洞检查`, `GDPR` | "检查这段 JWT 鉴权代码是否存在安全漏洞，并给出加固建议。" |
| **Office 助手** | `读 Word`, `写 Excel`, `PDF 提取` | "从这份 PDF 报告中提取所有的表格数据并转换成 CSV 格式。" |

---

## 💡 使用小贴士

1. **直接指名道姓**: "作为 **03 号前端专家**，..." 权重最高。
2. **场景唤醒**: 只要你的描述包含上述关键词，Trae 就会自动关联对应的 `SKILL.md`。
3. **组合技**: 先让 **PM** 梳理需求，再让 **架构师** 设计 API，最后让 **开发** 实现代码。
